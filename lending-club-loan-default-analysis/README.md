# Lending Club Loan Default Analysis (Excel Dashboard)

## Overview

This project analyzes loan performance and borrower risk using a dataset of 9,578 Lending Club loan records. The objective is to identify key factors that influence loan default and visualize the findings using an interactive Excel dashboard.

## Key KPIs

* Total Loans: 9,578
* Overall Default Rate: 16%
* Average Interest Rate: 12%
* Average FICO Score: (from dashboard)

## Key Insights

* Small business loans have the highest default rate at 27.79%.
* Borrowers with low FICO scores default at 32.03%, compared to 12.30% for high FICO borrowers.
* Default risk increases rapidly as interest rates rise, peaking at 35.14% in the 20%â€“25% range.
* Medium revolving credit utilization records the highest default rate at 41.63%.
* Loans that met the credit policy still account for 80.50% of total defaults.
